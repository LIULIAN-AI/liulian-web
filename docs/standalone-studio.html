What is a dataset?

A LIULIAN dataset is a folder of time-series shards plus a YAML manifest that pins the schema, topology, and an integrity hash. The manifest is the contract; the shards are interchangeable.

Click any row to inspect the manifest, see field types with example values, and the SHA-256 of the data behind this version of the contract.

⌘K → "new dataset"

What is a run?

A run is one training trial. Created by an experiment (which packages config + hyperparameter search). Compare them in the table to the left; ⌘O on the focused row to open the run's loss curves and metrics.

Status uses typography, not colour: italic is running, bold is failed.

⌘K → "compare runs"

What is a forecast?

A forecast is a triplet: a model, a dataset, and a future window. It carries the predicted mean and Q05–Q95 prediction intervals at every horizon step.

Open the BI canvas (/forecast) to overlay multiple forecasts on the same station, or compare a model against Chronos-2 zero-shot for an apples-to-apples baseline.

⌘K → "open in BI canvas"

What is a report?

A report is a saved BI canvas layout: panels + filters + an optional public-share slug. It freezes a question in a way that anyone with the URL can re-open.

Build one in /forecast, then "Save report" from the toolbar. Public reports are read-only; the same URL renders both the live data and a PDF on demand.

⌘K → "export PDF"
